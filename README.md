Welcome to your new dbt project!!

### Using the starter project

Try running the following commands:
- dbt run
- dbt test
- dbt debug

### Documentation:
For documentation you first need to run the following command:
- dbt docs generate
After the docs are generated you need to run the following command to start server (this will run an app on localhost):
- dbt docs serve
